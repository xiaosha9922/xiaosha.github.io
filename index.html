java 连接ftp客户端 切换目录失败（linux）（原因:ftp连接用户主目录错误）

解决办法：

1，修改用户ftp根目录。

a: /home/etc 目录下找到文件 passwd
b: 搜索用户如：test 结果如下：test:x:1005:1005::/home/teamwork/data:/bin/bash 修改根目录

2，锁定用户主目录。

a：/home/etc/vsftpd 目录下找到文件 vsftpd.conf
b: 设置chroot_local_user=YES allow_writeable_chroot=YES

linux防火墙开通端口

查看已开放端口
more /etc/sysconfig/iptables
1，开启8000端口
/sbin/iptables -I INPUT -p tcp –dport 8000 -j ACCEPT
2，保存配置
/etc/rc.d/init.d/iptables save
3，重启服务
/etc/rc.d/init.d/iptables restart
